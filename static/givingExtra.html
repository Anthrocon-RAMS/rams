Kicking in Extra Money

Want to give us a little extra cash to help make the festival better? If you do, we'll throw in a few special keepsakes! Each donation level gets you an extra bit of swag, in addition to all of the swag from all of the lower donation levels. None of these things (except the tshirt) will be for sale, so this is the only way to get them!

If you've registered as a Supporter in the past, that's still available as a donation level. It's pretty much the same as always, except that now you can donate even more if you'd like, and we'll even kick in a few extra mementos as thanks.

We also let you identify an affiliate; basically, this tells us which groups are throwing the most financial support into MAGFest so that we can reciprocate by doing cool things for those groups.
